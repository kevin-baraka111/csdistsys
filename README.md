# Simple Client-Server Application

## Description

This project is a simple client-server application developed using PHP, Apache, and MySQL on a localhost environment. It demonstrates communication between the client, server, and database using a layered architecture.

## Technologies Used

- PHP
- Apache (XAMPP)
- MySQL
- HTML
- Bootstrap

## Project Structure

- `insert.php` – Displays the form and inserts data into the database.
- `index.php` – Displays all saved records.
- `dbconnect.php` – Connects the application to the MySQL database.

## How to Run

1. Start Apache and MySQL in XAMPP.
2. Create a database named `csdistsys`.
3. Import or create the `users` table.
4. Place the project folder inside the XAMPP `htdocs` folder.
5. Open `http://localhost/csdistsys/insert.php` in your browser.
